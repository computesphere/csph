# csph — ComputeSphere CLI

Public distribution of release binaries for the **ComputeSphere CLI** (`csph`).

## Install

**Homebrew**

```sh
brew install computesphere/cli/csph
```

**Direct download**

Grab the archive for your platform from the
[latest release](https://github.com/computesphere/csph/releases/latest),
extract it, and put `csph` on your `PATH`.

## Documentation

See the [ComputeSphere CLI docs](https://docs.computesphere.com/docs/cli).

---

This repository hosts release archives only — it is the download target for the
Homebrew formula and direct downloads. Report issues and feedback through the
ComputeSphere console and support.
